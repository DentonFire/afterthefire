# afterthefire
After the Fire: Recovery Guide

An interactive, mobile-friendly recovery guide for residents of Denton, Texas, affected by a fire.

Overview

This website serves as a centralized hub of critical resources, contact information, and step-by-step guidance for individuals and families navigating the immediate aftermath of a fire. It is designed to be accessible, easy to navigate on mobile devices, and provides practical tools for recovery.

Key Features

Interactive Recovery Roadmap: Clear, actionable steps for the first 24 hours, handling finances, replacing vital documents, and managing utilities.

Incident Information Generator: A secure, client-side tool that allows residents to input their case details (incident number, investigator name, insurance info) and generate a downloadable PDF for their records. No data is stored or transmitted.

One-Touch Contact: Direct "click-to-call" buttons for local emergency services, utilities, and aid organizations (Red Cross, Salvation Army).

Mobile-First Design: Optimized for easy reading and navigation on smartphones, with instructions for saving the guide to the home screen as a web app.
